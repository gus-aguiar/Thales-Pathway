## 🚀 Como usar

### 1. Instalar dependências
```bash
npm install
```

### 2. Rodar os testes

**Modo interativo (recomendado):**
```bash
npm run cypress:open
```
Depois clique em `project.spec.js` ou `bonus.spec.js`

**Modo terminal:**
```bash
npm test
```

## 💡 Dicas para modificar

1. **Personalize o conteúdo**: Edite o `index.html` e coloque suas próprias lições aprendidas
2. **Mude o estilo**: Edite o `style.css` para deixar com sua cara
3. **Adicione sua foto**: Substitua a URL da imagem no `id="minha_foto"`
4. **Teste sempre**: Depois de cada modificação, rode `npm test` para verificar se ainda está passando

## ⚠️ Cuidados importantes

- **NÃO mude os nomes dos IDs**: Os testes dependem deles (`cabecalho`, `titulo`, `minha_foto`, etc.)
- **Contagem de caracteres**: O artigo precisa ter 300-600 caracteres e o aside 100-300 caracteres
- **Semântica**: Mantenha as tags HTML semânticas (`header`, `footer`, `article`, `aside`)
