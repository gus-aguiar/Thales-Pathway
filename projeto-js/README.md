# 🎯 Projeto JavaScript - Desafios de Programacao

Ola! Bem-vindo ao projeto de desafios de JavaScript do Thales.

Este projeto contem uma serie de exercicios para praticar logica de programacao e JavaScript basico.

## 🚀 Como comecar

### 1. Instale as dependencias

```bash
npm install
```

### 2. Execute os testes

```bash
npm test
```

## 📁 Estrutura do projeto

```
projeto-js/
├── src/
│   ├── challenges.js   # Desafios 1 a 10
│   └── challenges2.js  # Desafios 11 a 13
├── tests/              # Arquivos de teste
├── package.json        # Dependencias e scripts
└── README.md           # Este arquivo
```

## 📝 Desafios

### Desafios Principais (challenges.js)

1. **compareTrue** - Crie uma funcao usando o operador &&
2. **calcArea** - Calcula a area de um triangulo
3. **splitSentence** - Divide uma frase em array de palavras
4. **concatName** - Concatena strings de um array
5. **footballPoints** - Calcula pontos de um campeonato de futebol
6. **highestCount** - Conta repeticoes do maior numero
7. **catAndMouse** - Jogo de gato e rato
8. **fizzBuzz** - Classico desafio de divisibilidade
9. **encode/decode** - Codifica e decodifica vogais
10. **techList** - Cria lista de tecnologias

### Desafios Bonus (challenges2.js)

11. **generatePhoneNumber** - Gera numero de telefone formatado
12. **triangleCheck** - Verifica se pode formar um triangulo
13. **hydrate** - Calcula copos de agua para beber

## 🧪 Como testar

### Rodar todos os testes

```bash
npm test
```

### Rodar um teste especifico

```bash
npm test compareTrue
npm test calcArea
# etc...
```

## 🔧 Linter (Opcional)

Para verificar o estilo do codigo:

```bash
npm run lint
```

> Nota: O linter nao sera avaliado, mas pode ajudar a manter um codigo limpo!

## 💡 Dicas

- Leia atentamente cada requisito antes de implementar
- Use `console.log()` para testar localmente, mas remova antes de finalizar
- Nao altere o nome das funcoes existentes
- Os parametros podem ser alterados se necessario
- Voce pode criar funcoes auxiliares se precisar
